Spotify Party Quiz Game

> Hackathon, HackNotts 21, Feb 2022
CSS3 HTML5 JavaScript MySQL PHP

Inspiration

Drawing from mobile party games and platforms like Kahoot, we created a music-based quiz that reveals friends' music tastes. Using Spotify's API, it turns personal listening data into hilarious, revealing trivia questions about everyone's guilty pleasures.

What it does

A 4-player multiplayer game where participants connect their Spotify accounts. The app aggregates everyone's music data to generate personalized multiple-choice questions like "Who has been listening to The Beatles most recently?" or "What's Adam's current favorite song?"

How we built it

Built with PHP backend, HTML/CSS/JavaScript frontend, and MySQL database. Spotify's API provides user data including top tracks, albums, and artists. The system manages game lobbies, players, and custom question generation.

Challenges Faced

  • Spotify API developer mode limitations requiring pre-approved user emails
  • Server configuration challenges with security restrictions
  • Client synchronization issues affecting questions and scoreboards

Features we're proud of

  • Professional, responsive UI that works beautifully on all devices
  • Rich Spotify integration with album covers and music embeds
  • Multi-lobby system allowing simultaneous games with unique codes

Future of the Game

  • Expand to more question types and themed rounds
  • Support for 3-8 players with flexible answer options
  • Mobile apps using Spotify SDK for automatic music playback

Note

Due to Spotify API limitations, user emails must be pre-approved by our team. The game requires prior arrangement to run.
